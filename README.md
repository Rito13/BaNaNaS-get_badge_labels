# Description 
This repository is an automated documentation of various labels used in grfs uploaded to BaNaNaS. Currently only following are supported:
 - Badge labels
 - Cargo labels

The information is updated 3 times per week (Tuesday, Friday and Sunday). Due to continuous testing and development, all changes must be manually approved, therefore the update time is not fixed. Pending changes can be viewed in pull requests tab. If you find any mistakes, please create appropriate issue.
